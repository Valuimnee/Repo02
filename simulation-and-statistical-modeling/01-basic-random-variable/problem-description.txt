Краткое описание:
Реализация датчиков базовой случайной величины(БСВ) по мультипликативному конгруэнтному методу и методу Макларена-Марсальи.
Проверка точности моделирования БСВ на тестах совпадение моментов, ковариация, равномерность двумерного распределения.
Определение периодов сгенерированных последовательностей.

Постановка задачи:
1. Осуществить моделирование n = 1000 реализаций БСВ с помощью мультипликативного конгруэнтного метода (МКМ) 
с параметрами a0= a01=261463909, β = max{c1, M – c1}, c1=474379977, M = 2^31 
и вывести 100-ый, 900-ый и 1000-ый элементы сгенерированной последовательности. 
2. Осуществить моделирование n = 1000 реализаций БСВ с помощью метода Макларена-Марсальи, 
используя в качестве простейших датчиков БСВ датчики D1 – датчик из первого задания, D2 – датчик по методу МКМ 
с параметрами a0= a02 =234289925, β = max{c2, M – c2}, c2=3097871, M = 2^31, K =192 – объем вспомогательной таблицы 
и вывести 100-ый, 900-ый и 1000-ый элементы сгенерированной последовательности.
Для каждого из построенных генераторов:
3. Проверить точность моделирования с помощью теста «совпадения моментов» с уровнем значимости ε = 0.05.
4. Проверить точность моделирования с помощью теста «ковариация» с уровнем значимости ε = 0.05. 
В качестве параметра t выбрать значение 30. Вывести все такие значения лага, при котором тест не проходит.
5. Проверить точность моделирования с помощью теста «равномерность двумерного распределения» с уровнем значимости ε = 0.05. 
Параметр k выбирать самостоятельно.
6. Определить длину периода выходной последовательности для генератора.
